# ML Experimentation Platform Backend

This directory contains the Python backend for the machine learning experimentation web application.

## Setup and Running

1.  **Install dependencies:**
    ```bash
    pip install -r requirements.txt
    ```

2.  **Run the FastAPI server:**
    ```bash
    uvicorn app.main:app --reload
    ```

The server will be running at `http://127.0.0.1:8000`.
